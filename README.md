Maths Mobile App (Expo)

Quick demo mobile app for practicing basic arithmetic.

Prerequisites
- Node.js
- npm or yarn
- Expo CLI (optional) — not required if using `npx`

Run locally

```bash
cd "c:/Users/Admin/Documents/Mobile App"
npm install
npx expo start
```

Open the app using the Expo Go app on your phone, or run on an emulator.

Files
- `App.js` — root app and simple navigation
- `components/Quiz.js` — quick arithmetic quiz
- `components/Practice.js` — practice operations
- `components/Calculator.js` — small calculator
